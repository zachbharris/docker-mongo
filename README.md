## Helpful notes

- Start and stop

  ```
  // start
  docker-compose up

  // rebuild and start
  docker-compose up --build

  // remove container, images and networks
  docker-compose down
  ```

  - [official docker docs on compose cli](https://docs.docker.com/compose/reference/overview/)
